# Inder's 2026 Roadmap

6-month skill acquisition plan, built for visible accountability.

**Target:** AI-adjacent Senior QA / QE role, remote or Ontario-based.
**Cadence:** 2-week ship cycles.
**Started:** 2026-06-08 (Cycle 1)
**Plan target completion:** 2026-11-22 (end of Cycle 12)

## How this works

- Each cycle is 2 weeks long.
- Each cycle ships ONE visible artifact — a public repo, a Claude Code skill, a passed exam, etc.
- A scheduled Claude routine runs every Sunday 7 PM and posts a status issue here.
- Slipped cycles are marked `[~]` and stay in the history. Hiding slips is worse than slipping.
- 2 grace cycles per 6 months. Using one = no recorded slip, no protocol triggered.

Full design and failure protocol live in the spec (private repo).
